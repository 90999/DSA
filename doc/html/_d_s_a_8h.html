Detailed Description

Library header for sending and receiving commands using the DSA protocol.

The high-end Compact Disc modules made by daisy-laser http://www.daisy-laser.com are controlled by the DSA protocol. A three wire interface using a data- (DSA), strobe- (STB) and acknowledge (ACK) line. More info on the DSA protocol on http://www.daisy-laser.com/technology/techdsa/techdsa.htm
