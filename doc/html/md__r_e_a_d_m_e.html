DSA

Arduino library for the CD-Pro2 DSA protocol

The high-end Compact Disc modules (CD-Pro2, CD-Pro2M and CD-Pro2) made by daisy-laser are controlled by this DSA protocol. A three wire interface using a data- (DSA), strobe- (STB) and acknowledge (ACK) line. Read more about the DSA protocol

This is a small piece of work (in progress) to convert the DSA protocol assembler code from my CDP-1 project into a simple library to control a CD-Pro2 with an Arduino board. Just for fun!

Installation

Download the latest version and unzip (atleast) the lib directory. Copy the unziped directory called "DSA" to the Arduino libraries folder. For OSX this is __~/Documents/Arduino/libraries/__ on Windows __My Documents. More info about adding libraries to your Arduino IDE can be found here.

Examples

SimpleCD

Control the CD-Pro unit by serial commands

Ard2Ard

Besides controlling a CD module this library can also be used to communicate between two Arduino boards, check the Ard2Ard example.
