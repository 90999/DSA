bool DSA::waitForMessage ( byte *  opcode,
byte *  parameter 
)

Wait for a message to receive.

Deprecated:
Try to avoid using this function. It will block your program for the timeout time when no message is received. Beter keep checking transmitRequested() in you main loop.

When a message is expected this function will wait for the message to arrive. A timeout is set to abort this function at timeout.

Parameters
*opcodePointer to the opcode variable that will updated after a succesful transfer
*parameterPointer to the parameter variable that will updated after a succesful transfer.
Returns
True on a succesful transfer, False on a timeout or communication error
